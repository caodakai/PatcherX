## 自定义规则
### 1. 导出的时候选择文件夹，则导出所有文件

### 2. 导出时，如果选择了文件夹，同时又选择了"子文件"，那么不会导出文件夹所有内容，只会导出"子文件" 
###### (注：子文件包括子文件夹)
